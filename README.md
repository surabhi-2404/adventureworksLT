## Description

This project is an interactive Power BI dashboard designed to analyze and visualize data for better decision-making. 
The report provides insights through dynamic charts, filters, and KPIs, helping users explore trends, patterns, and performance metrics efficiently.

## Features

Interactive dashboard visuals<br>
KPI cards for quick insights<br>
Dynamic filters and slicers<br>
Clean and structured data model<br>
Drill-down analysis capability<br>
User-friendly layout<br>
